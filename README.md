# minitables
DataTables equivalent, dependent on Bootstrap 5, but not on jquery.

## Usage:
- ### JsonMinitable
  Any well formatted json can be displayed with minitables.
  HTML:
  ```
   <div id="table_container">
            <table class="table table-striped table-hover" id="minitable">
                <thead>
                </thead>
                <tbody>
                </tbody>
            </table>
    </div>
  ```
